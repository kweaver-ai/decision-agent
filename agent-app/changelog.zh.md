# 版本 changelog 说明

## 0.1.0

### 核心功能
- 会话管理：自定义标题、已读/未读状态、进行中对话标识
- Agent 对话：终止、恢复、重新生成、编辑问题
- Agent 调试运行与 API 能力
- 文件上传全文内容存储进度检查

### 性能优化
- 基于 IP 的会话亲和性，支持多 Pod 部署
- 对话吐字优化：有缓冲 channel + sonic 序列化
- 首 Token 响应优化，支持 V2 版本 Executor

### 可观测性
- 关键链路 trace/log 埋点
- Agent Run 成功/失败日志上报
- 轨迹分析服务（Session/Run 列表及详情）

### 服务能力
- Agent 权限对接（应用账号/业务账号）
- 业务域支持
- Redis 缓存集成
- Session 统一生命周期管理
- 服务器优雅关闭
- SQL 脚本下沉到微服务内部
