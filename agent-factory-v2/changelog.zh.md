# 版本 changelog 说明

## 0.1.0

### 核心功能
- Agent 配置、发布、模板、复制
- Agent 自定义空间管理
- AI 生成接口（使用默认大模型）
- Agent 批量获取指定字段接口
- 策略分类与策略列表接口

### 配置能力
- 工具超时时间配置
- 任务规划模式开关
- 知识条目数量限制校验
- Agent self_config 字段结构查询
- 数据源配置（指标、知识条目）

### 权限与业务域
- 业务域关联功能
- Agent 权限检查（应用账号/业务账号）
- Agent 三态状态过滤

### 运维增强
- HTTP Server 优雅关闭
- HTTP 请求日志（客户端/服务端区分）
- MQ 配置与审计日志处理
- SQL 脚本下沉到微服务内部
- Redis/MQ 依赖声明
